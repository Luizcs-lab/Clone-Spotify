# 🚀 CloneSpotify
Página web simples desenvolvida com intuito de fortalecer os conhecimentos em front-end, as boas práticas e organização das pastas de trabalho, utilizando framework bootstrap para responsividade.  

## 📌 Sobre o projeto

O **CloneSpotify** é uma aplicação realizada utilizando os conhecimentos de html, css e bootstrap com objetivo de praticar a estrutução de páginas que se adaptam as diferentes resoluções de dispositivos.

## 🛠️ Tecnologias utilizadas

* **HTML5** → Estrutura da aplicação
* **CSS3** → Estilização da interface
* **Bootstrap** → Design responsivo e componentes visuais
